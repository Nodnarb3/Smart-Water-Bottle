## Build:
1) Clone/download the repository
2) Open the project in the Arduino IDE
3) Click Verify in the upper-left hand corner

## Run:
1) Plug the smart water bottle into your computer
2) Click Upload in the upper-left hand corner of the Arduino IDE
3) Once code is uploaded to the smart water-bottle, anytime it is plugged into a power source, the code will automatically execute
